NETSNOOP

OFFENSIVE DEFENSIVE

DESCRIPTION

Advanced packet sniffer written in Python that captures and decodes live network traffic, supporting multiple protocols including IPv4, ARP, IPv6, TCP, UDP, ICMP, DNS, DHCP, and VLAN.

FEATURES

  • Real-time protocol statistics
  • Hex dump output
  • PCAP format saving
  • Customizable filters
